## 🎫 HOW TO BOOK A TICKET (User Module Test)

1. **Login as User** → http://localhost:5173/user
2. **Browse Events** → See 25+ events from across India
3. **Use Filters**:
   - Search by name/city/state
   - Select category
   - Toggle "Free Events Only"
4. **Click "Book Now"** on any event
5. **Select Tickets** → Use +/- buttons
6. **View Price Breakdown**:
   - Base price
   - Convenience fee
   - GST
7. **Proceed to Payment**
8. **Choose Method** → UPI/Card/Net Banking/Wallet
9. **Enter Details** → Any dummy data works
10. **Click "Pay"** → Get instant confirmation!
11. **Check Console (F12)** → See booking confirmation email!

---

## 🎪 HOW TO ADD EVENT (Organizer Test)

1. **Signup as Organizer** → Select role "Organizer"
2. **Login** → Access organizer dashboard
3. **Click "Create Event"**
4. **Fill Details**:
   - Title & Description
   - **Select State** → Choose from 22 states dropdown
   - City → Enter city name
   - Venue → Location
   - Date & Time
   - Category → Select type
   - Total Tickets → Number
   - Price → OR check "This is Free Event"
   - Image URL → Use Unsplash link
5. **Click "Create Event"**
6. **Event Added!** → Appears in your dashboard
