## 💰 **How to Book Tickets**

1. Click **"Book Now"** on any event card
2. Select number of tickets (+ / - buttons)
3. View price breakdown
4. Click **"Proceed to Payment"**
5. Choose payment method (any option)
6. Enter dummy card details
7. Click **"Pay [amount]"**
8. ✅ Get confirmation with Booking ID!

## 🛠️ **Quick Fix if Issues Occur**

**Problem:** User module not working  
**Fix:** 
1. Refresh page (Ctrl + F5)
2. Check you're logged in
3. Clear browser cache
4. Use correct role when signing up

**Problem:** No events showing  
**Fix:**
1. Remove all filters
2. Set category to "All"
3. Uncheck "Free Events Only"
4. Clear search box

**Problem:** Can't book  
**Fix:**
1. Check ticket availability
2. Try free event first
3. Ensure logged in as User
